# commit-helper

Generate conventional commit messages and pull requests with approval workflow.

## Skills

- **committing-staged-with-message** — Generate commit message for staged changes, pause for approval, then commit
- **creating-pr-from-branch** — Analyze branch commits, generate PR title+body from template, push and create PR

## Install

```bash
claude plugin install commit-helper@qte77-claude-code-utils
```
