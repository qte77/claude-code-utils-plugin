# rust-dev

Rust implementation, testing, and code review skills, deploys cargo permissions via SessionStart hook.

## Skills

- **implementing-rust** — Implements concise, streamlined Rust code matching exact architect specifications
- **testing-rust** — Writes tests following TDD (using cargo test, proptest, and insta) best practices
- **reviewing-rust** — Provides concise, focused Rust code reviews matching exact task complexity requirements

## Deployed files

- **settings.local.json** -> `.claude/settings.local.json` — cargo permissions (if cargo detected, copy-if-not-exists)

## Install

```bash
claude plugin install rust-dev@qte77-claude-code-plugins
```
