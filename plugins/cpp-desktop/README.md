# cpp-desktop

C++ desktop application development with wxWidgets, GTK, and Qt frameworks.

## Skills

- **implementing-cpp** — Implements C++17 desktop GUI code with framework-specific patterns (wxWidgets, GTK3, Qt)
- **reviewing-cpp** — Reviews C++ desktop code for memory safety, framework anti-patterns, and thread safety
- **analyzing-cpp-codebase** — Analyzes C++ codebase architecture, dependency graphs, and framework detection

## Install

```bash
claude plugin install cpp-desktop@qte77-claude-code-plugins
```
