# go-dev

Go implementation, testing, and code review skills, deploys go tool permissions via SessionStart hook.

## Skills

- **implementing-go** — Implements concise, streamlined Go code matching exact architect specifications
- **testing-go** — Writes tests following TDD (using go test, testify, and rapid) best practices
- **reviewing-go** — Provides concise, focused Go code reviews matching exact task complexity requirements

## Deployed files

- **settings.local.json** -> `.claude/settings.local.json` — go tool permissions (if go detected, copy-if-not-exists)

## Install

```bash
claude plugin install go-dev@qte77-claude-code-plugins
```
