# docs-generator

Academic writeup generation, PRD-to-JSON conversion, and interactive user story builder.

## Skills

- **generating-writeup** — Generates academic/technical writeups with IEEE citations and pandoc PDF output
- **generating-prd-json-from-prd-md** — Generates prd.json task tracking file from PRD.md requirements document
- **generating-interactive-userstory-md** — Interactive Q&A to build UserStory.md from user input

## Install

```bash
claude plugin install docs-generator@qte77-claude-code-utils
```
